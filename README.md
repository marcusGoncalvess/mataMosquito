# mataMosquito
Um jogo simples feito apenas com JavaScript onde ficará aparecendo mosquitos na tela e o objeto é o jogador dar um click
sobre o mosquito. Cada dificudade tem um tempo para o mosquito desaparecer, se o mosquito desaparecer sem se o jogador ter clicado
nele,isso irá dirar uma das 3 vidas do jogador. Cada dificuldade tem um tempo, se esse tempo for alcançado sem perder todas as
vidas o jogador vence.
